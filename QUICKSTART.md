# 快速開始指南

## 5分鐘快速上手

### 第一步：安裝套件
```bash
pip install yfinance pandas matplotlib
```

或使用 requirements.txt：
```bash
pip install -r requirements.txt
```

### 第二步：啟動程式
```bash
python main.py
```

### 第三步：先看看大盤

1. 點選「查看大盤」
2. 觀察台股加權指數的走勢
3. 嘗試切換不同時間範圍和圖表類型
4. 點選「返回」回到主畫面

### 第四步：建立第一個分類

1. 點選「管理分類」
2. 在「新增分類」欄位輸入 `科技股`
3. 點選「新增」
4. 點選「返回」

### 第五步：新增第一支股票

1. 點選「新增股票」
2. 輸入 `2330`（只要輸入數字即可，系統會自動判斷上市/上櫃）
3. 勾選「科技股」分類
4. 點選「新增」
5. 等待系統自動嘗試並下載資料（會顯示 `2330.TW` 或 `2330.TWO`）

### 第六步：查看技術分析

1. 點選「瀏覽股票」
2. 選擇「科技股」或「全部股票」
3. 點選台積電的「技術分析」按鈕
4. 嘗試切換不同的圖表和時間範圍

## 推薦練習股票

建議先新增以下幾支股票來熟悉系統：

- **2330**（台積電）- 半導體龍頭
- **2317**（鴻海）- 電子製造
- **2454**（聯發科）- IC設計
- **2412**（中華電）- 電信股
- **2882**（國泰金）- 金融股
- **0050**（元大台灣50）- ETF

建議先建立分類（如「科技股」、「金融股」、「ETF」）再新增股票。

## 常用操作流程

### 建立自己的觀察清單

1. 新增分類「觀察清單」
2. 新增感興趣的股票
3. 將股票加入「觀察清單」分類
4. 定期點選「更新所有股價」

### 比較多支股票

1. 依分類瀏覽股票
2. 逐一點選「技術分析」
3. 使用「返回」按鈕快速切換

### 追蹤特定產業

1. 建立產業分類（如「半導體」、「金融」）
2. 將相關股票加入分類
3. 透過分類篩選快速查看同產業股票

## 常見問題快速解答

**Q: 如何新增上櫃股票？**
A: 輸入代碼時加上 `.TWO`，例如 `5274.TWO`

**Q: 資料多久更新一次？**
A: 手動更新。建議每日收盤後點選「更新所有股價」

**Q: 如何備份資料？**
A: 複製 `database/taiwan_stock.db` 檔案即可

**Q: 程式執行錯誤怎麼辦？**
A: 
1. 確認已安裝所有必要套件
2. 檢查網路連線
3. 查看終端機的錯誤訊息

## 進階使用技巧

### 技巧 1：批次新增股票
可以連續新增多支股票而不關閉視窗，系統會自動清空輸入框。

### 技巧 2：使用多重分類
一支股票可以同時屬於多個分類，例如台積電可以同時是「電子股」、「權值股」、「觀察清單」。

### 技巧 3：時間軸導航
在技術分析頁面，使用「◀ 上一段」和「下一段 ▶」可以快速查看歷史資料。

### 技巧 4：MA均線判斷趨勢
- **多頭排列**：股價 > MA20 > MA60
- **空頭排列**：股價 < MA20 < MA60
- **黃金交叉**：MA20向上穿越MA60
- **死亡交叉**：MA20向下穿越MA60
